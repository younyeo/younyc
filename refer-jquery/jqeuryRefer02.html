기본 선택자
종류 제이쿼리 예시 자바스크립트 설명
필터링 .eq() 👍🏻 $("selector").eq(3) 선택한 요소의 인덱스 번호에 해당하는 요소를 찾습니다.
.filter() $("selector").filter() 속성은 흐림 효과나 색상 변형 등 그래픽 효과를 요소에 적용합니다. 보통 필터는 이미지, 배경, 테두리 렌더링을 조정하는 데 쓰입니다.
.first() $("selector").first() first 의사 클래스는 @page @-규칙과 함께 사용되며, 출력 시의 첫 페이지를 나타냅니다.
.has() 👍🏻 $("selector").has() .has()로 특정 요소를 가지고 있는 요소를 선택할 수 있습니다.
.is() $("selector").is() is() 메소드는 특정요소가 선택요소와 관련값 또는 현재의 상태 등을 확인 후 Boolean 값 true, false로 반환합니다. 그래서 is()를 사용해 원하는 선택자 결과... 원하는 태그인지 확인할 때 유용하죠.
.last() $("selector").last() 선택된 요소에서 마지막 번째 요소를 찾습니다.
.map() $("selector").map() HTML map 요소는 area 요소와 함께 이미지 맵(클릭 가능한 링크 영역)을 정의할 때 사용합니다.
.not() $("selector").not() .not()은 선택한 요소 중 특정 선택자를 제외한 요소를 선택합니다.
.slice() $("selector").slice() .slice()는 일치하는 요소의 일부분만 선택합니다.
트리구조 .children() $("selector").children() .children()은 어떤 요소의 자식 요소를 선택합니다.
.closets() $("selector").closets() 선택한 요소를 포함하면서 가장 가까운 상위 요소를 선택합니다.
.find() 👍🏻 $("selector").find() .find()는 어떤 요소의 하위 요소 중 특정 요소를 찾을 때 사용합니다.
.next() 👍🏻 $("selector").next() 선택한 요소의 다음 요소를 선택합니다.
.nextAll() $("selector").nextAll() 선택한 요소의 다음 모든 형제 요소를 선택합니다.
.nextUntil() $("selector").nextUntil() 선택한 요소의 다음에 위치한 요소를 조건에 맞을 때까지 찾습니다.
.parent() 👍🏻 $("selector").parent() parentNode .parent()는 어떤 요소의 부모 요소를 선택합니다.
.parentUntil() $("selector").parentUntil() 선택한 요소의 부모 요소 중 조건에 맞을 때까지 찾습니다.
.prev() 👍🏻 $("selector").prev() 선택한 요소의 이전 요소를 선택합니다.
.prevAll() $("selector").prevAll() 선택한 요소의 이전 모든 형제 요소를 선택합니다.
.prevUntil() $("selector").prevUntil() 선택한 요소의 이전에 위치한 요소를 조건에 맞을 때까지 찾습니다.
.siblings() $("selector").siblings() 선택한 요소를 제외한 형제 요소를 모두 찾습니다.
기타 .add() $("selector").add() .add()는 어떤 요소를 추가로 선택할 때 사용합니다.
.addBack() $("selector").addBack() .addBack()은 현재 선택한 요소와 함께 이전에 선택한 요소도 선택하게 합니다.
.contents() $("selector").contents() .addBack() 메서드는 현재 선택된 요소의 이전 요소를 한번 더 찾아주는 필터링 역할의 메서드입니다.
.each() 👍🏻 $("selector").each() forEach jQuery를 사용해 배열을 관리하고자 할 때 each() 메서드를 사용할 수 있습니다. each() 메서드는 매개 변수로 받은 것을 사용해 for in 반복문과 같이 배열이나 객체의 요소를 검사할 수 있는 메서드입니다.
.end() $("selector").end() .end() 메서드는 현재 선택된 요소의 이전 요소를 선택하는 메서드입니다.
